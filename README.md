## Project

### Problem
Briefly describe the real-world problem your project solves.

Example:
Monitoring crowded public places manually is difficult and inefficient. Security teams often fail to detect suspicious activities such as weapon presence, loitering, or overcrowding in real time.

### Solution
Describe your project solution.

Example:
We built an **AI-powered surveillance system** that uses computer vision and deep learning to detect suspicious activities such as crowd overload, loitering, weapon detection, and criminal identification in real time. The system also ensures **privacy protection** by applying automatic face blurring and stores alert clips securely in the cloud.
